# Global_fashion_retail_powerbi

The project uses a synthetic global fashion retail dataset generated through rule-based simulation to replicate real-world transactional patterns.
It contains over 4 million sales records across multiple tables (sales, products, customers, stores, promotions) and covers a two-year period.
The dataset incorporates temporal effects, geographic hierarchies, location-based pricing, and product category structures, making it suitable for large-scale BI modeling and analysis.
